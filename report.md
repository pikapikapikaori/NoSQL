# 10195101485 胡泓震

## 数据库设计

数据库分为三种节点, Station, Line, 和Run.

一个Station节点代表一个车站, 具有english, name和id三个属性.

### Station节点实例

```json
{
  "identity": 19887,
  "labels": [
    "Station"
  ],
  "properties": {
"english": "YongTongLu",
"name": "永通路",
"id": "41394"
  }
}
```
